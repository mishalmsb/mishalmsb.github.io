Friendly Bets:
Project Two @ GA WDI
2016

The Euro 2016 was about to start, so I made an betting website where I as an admin will create games and any other users can start a bet on that game.

This is a full-stack MVC web application in Ruby on Rails with PostgreSQL.

More information can be found on the project.

use email: mishalmsb@gmail.com password: mishal to login as Admin

Some of the technologies used:

  • Ruby on Rails
  • PostgreSQL
  • CarrierWave
  • Devise
  • Bootstrap
  • jQuery
  • ActionMailer
