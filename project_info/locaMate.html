LocaMate:
Project three @ GA WDI
2016

This was a group project and it was amazing, I did learn a lot and had a lot of fun during this week.

The idea of this project was to user google maps api to save locations, get directions, get friends current location and to be able to see a friend journey and track him.

Some of the technologies used:

  • Google Maps API
  • MongoDB
  • Node.js
  • Express.js
  • Bcrypt
  • Socket.IO
  • Bower
  • Bootstrap
